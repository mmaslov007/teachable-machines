# The Sentient Magic Box

An interactive web-based game where a mysterious cardboard box responds to your gestures using machine learning-powered computer vision.

## How to Play

Start the camera and interact with the box using four gestures:

- **Show Your Face** - The box recognizes you and moves to the left
- **Peace Sign** - Make peace with the box and it dances downward
- **Hold Up a Phone** - The box reacts with sarcastic comments and moves right
- **Hide Everything (Blank)** - Make the box feel lonely

## Features

- Real-time gesture recognition using TensorFlow.js and Teachable Machine
- Smooth color transitions based on detected gestures
- Witty remarks from the box's personality
- Responsive design with centered camera feed
- Independent box movement that doesn't interfere with text

## Getting Started

1. Open `index.html` in a web browser
2. Click "Start Camera" to begin
3. Show gestures to the webcam to interact with the box
4. The box will respond with visual feedback and sarcastic commentary

## Technology

- TensorFlow.js - Machine learning in the browser
- Google Teachable Machine - Pre-trained gesture model
- Vanilla HTML/CSS/JavaScript
